# Neretva

## How to Run
### KIR
```bash
python kir.py --input [bam_path] > [out_path]
```

### CYP (Currently supports CYP2C8, CYP2C9, CYP2C19, CYP2D6)
```bash
python cyp.py [GENE] [bam_path] > [out_path]
```
